We consider a finite set of natural numbers represented as binary numbers shown in \refTab{two_bit_binary_numbers}. A value can be mapped to a \picalc{} process. \refLis{values_ABC_code} shows the \picalc{} implementation of the values 0,1,2,3 in ABC syntax. The keyword \textit{agent} defines a new processes. The process \textit{Zero} is modeled using alternative choice: it either receives a signal via the channel $a$ and switch off, or it receives two channels \textit{tt,ff} via \textit{a}, then it sends two signals via the channel \textit{ff}.
